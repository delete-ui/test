Папки с исходниками и результатами в виде png скринов хранятся по адресу test\x64\Debug.

Если вы хотите самостоятельно опробовать запустите командную строку с адреса исполняемого файла, test\x64\Debug,
введите имя исполняемого файла а затем через пробел путь к bmp файлу.

Пример:
 C:\test\x64\Debug>test.exe "C:\test\x64\Debug\исходные bmp"

P.S:
Если вам не устроил результат в строке попробуйте масштабировать cmd через колесико мышки и ctrl.
